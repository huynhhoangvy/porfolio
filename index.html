K-raoke

Capstone Project.

K-raoke is a webapp which fulfill your singing desire anywhere and anytime. The app is built with ReactJS front-end and Python - Flask back-end along with PostGreSQL database. K-raoke has some basic feature of a website like: register, login, logout, etc. Enjoy!

**Note: There's a side drawer storing selected songs where you can drag and drop to handle songs order.
